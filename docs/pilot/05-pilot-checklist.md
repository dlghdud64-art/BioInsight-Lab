## Phase C. 파일럿 후 정리

### 즉시 수행 (파일럿 당일)

- [ ] Operator 피드백 템플릿 수거
- [ ] Lead 피드백 템플릿 수거
- [ ] 관찰자 메모 수거
- [ ] 시드 데이터 리셋 (반복 파일럿 예정 시)

### 분석 (파일럿 후 1~2일)

- [ ] Blocker 건수 집계
- [ ] Friction 건수 집계
- [ ] Defer 건수 집계
- [ ] Blocker 항목 → V1.1 수정 항목 등록
- [ ] Friction 항목 → V1.1 또는 모니터링 분류
- [ ] Defer 항목 → V2 백로그 등록
- [ ] 결과 정리 문서 작성 (06-results-template.md)

### 판정

- [ ] Blocker 0건 → V1 릴리스 가능
- [ ] Blocker 있음 → 수정 후 재파일럿 일정 수립
- [ ] Friction N건 기준 — 수용 가능 수준인지 판단

## 파일럿 반복 시

1차 파일럿 후 Blocker 수정이 완료되면:
- [ ] Phase A 재수행 (시드 데이터 리셋 포함)
- [ ] 수정된 항목에 대해 집중 검증
- [ ] 2차 피드백 수거 및 비교
