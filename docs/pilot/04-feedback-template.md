# Ops Console V1 — Blocker / Friction / Defer 기록 템플릿

> 파일럿 중 발견한 문제를 아래 분류에 따라 기록한다.
> 분류 기준은 `console-v1-pilot-observation.ts`의 `PilotIssueClass`와 동일하다.

---

## 기록 규칙

1. 발견 즉시 기록 (파일럿 후 회상하면 누락됨)
2. 스크린샷 첨부 가능하면 첨부
3. "느낌"이 아니라 **구체적 상황** 기술 (어디서, 뭘 하려다, 어떻게 막혔는지)
4. 해결책 제안은 선택 사항 (문제 기록이 우선)

---

## 분류 기준

| 분류 | 코드 | 의미 | 조치 시점 |
|------|------|------|-----------|
| **Blocker** | `blocker` | 파일럿 진행 불가, 운영 흐름 중단 | V1.1 즉시 수정 (`v1.1_fix`) |
| **Friction** | `friction` | 진행 가능하나 혼란/지연 발생 | V1.1 또는 모니터링 |
| **Defer** | `defer` | 개선 가능하나 V1 범위 밖 | V2 이후 (`v2_defer`) |
