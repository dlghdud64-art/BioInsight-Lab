# Git 무한 로딩 방지 가이드

## 문제 원인
1. Git 에디터가 설정되지 않아 커밋 메시지 입력 대기 상태
2. PowerShell에서 `&&` 연산자 문제
3. Git 훅 실행 중 멈춤

## 해결 방법

### 1. 커밋 시 항상 메시지 직접 지정
```bash
git commit -m "커밋 메시지"
```

### 2. Git 에디터 비활성화 (이미 설정됨)
```bash
git config --global core.editor ""
```

### 3. 훅 건너뛰기 (필요시)
```bash
git commit --no-verify -m "커밋 메시지"
```

### 4. PowerShell에서 명령 실행
- `&&` 대신 세미콜론(`;`) 사용 또는 별도 명령 실행
- 예: `cd apps/web; npm install` 또는 두 개의 별도 명령

## 현재 설정 상태
- ✅ Git 에디터: 비활성화됨
- ✅ Git 훅: 활성화된 훅 없음 (샘플만 존재)
- ✅ 기본 브랜치: main
